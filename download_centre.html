Admin Instructions

How to Add Local Files:

  1. Organize Your Files:
    • Create folders in your project directory: "videos", "images", "audio"
    • Place your media files in these folders
    • Use descriptive filenames (e.g., "sunday_sermon_oct15.mp4")
  2. Update File Paths:
    • For videos: Use relative paths like "videos/filename.mp4"
    • For images: Use relative paths like "images/filename.jpg"
    • For audio: Use relative paths like "audio/filename.mp3"
  3. Edit This HTML File:
    • Open this file in a text editor (VS Code, Notepad++, Sublime, etc.)
    • Find the section you want to add to (Videos, Images, or Audio)
    • Copy one of the existing file blocks and edit it
    • Replace the file paths and information
